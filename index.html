LOL.GG

GIPHY SMASH


Giphy Smash is a web app that displays gifs of characters from the game Super Smash Bros, by using the giphy API. You can select or search characters and display their gifs accordingly. Gifs can be started and paused when clicked.

HTML, CSS, Javascript, Bootstrap, Giphy API
